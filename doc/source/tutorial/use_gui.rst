界面使用
============

基本介绍
------------

.. image:: ../_static/images/use_gui.png
    :align: center
    :scale: 50%

如上图，整个界面主要分为两个部分

- 上面黑色背景的这块是显示画面
- 下面带有各个动画标签的是时间轴

进度控制
~~~~~~~~~~~

| 按下空格键控制播放和暂停
| 如果播放到了末尾按下空格键，会从头播放

也可以在时间轴中左键拖动当前时刻，调整到你要查看的地方

时间轴视野控制
~~~~~~~~~~~~~~~~~~

你可以使用 “WASD” 来控制在时间轴上的视野：

.. list-table::

    *   -
        -   W 放大
        -
    *   -   A 左移
        -   S 缩小
        -   D 右移

.. raw:: html

    <del>有点像 FPS 游戏的键位</del>
